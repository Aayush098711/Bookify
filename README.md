# Overview

`Bookify` is a book exchange web app that allows users to easily exchange single or multiple books with others. Users can also message each other before completing an exchange, ensuring a smooth and efficient process.

# Main Features

1. Users can exchange single and multiple books with other people.
2. Chat with book owners to discuss details before exchanging.
3. Instantly AI-generated summaries for a quick overview of books.
4. Voice search to quickly find books using speech.
5. Audiobooks for users who prefer listening to book content.
6. Dynamic reviews and ratings on book detail pages to help users make informed decisions.


# Tech Stack

- **Framework:** Next.js
- **Styling:** Tailwind CSS
- **State Management:** React Context API
- **Package Manager:** pnpm
- **Language:** TypeScript
- **Hosting:** Vercel

# To Run Locally

```bash
# Clone the project
git clone https://github.com/your-username/pixor.git

# Go to the project directory
cd pixor

# Install dependencies
pnpm install

# Start the development server
pnpm dev

# The project will run at http://localhost:3000
```
